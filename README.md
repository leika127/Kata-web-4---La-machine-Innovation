# Kata-web-4---La-machine-Innovation

# Il faut créer 3 fichiers : 
- HTML 
- CSS 
- JS 

## Étapes de développement

Commit et Contenu:

- Mise en place des trois fichiers et liaisons HTML
- Structure du formulaire HTML 
- Zone de résultat + styles CSS
- Logique JavaScript : invocation de la créature 
- bonus* | Bouton "Renvoyer la créature"

## Description

Une page web interactive permettant d'invoquer sa créature de code personnelle.  
L'utilisateur remplit un formulaire (prénom + choix de créature), clique sur **Invoquer**, et le formulaire laisse place à un message d'invocation unique.

## Les créatures disponibles

- le Chat-Octo
- le Dragon-Debug
- la Licorne-CSS
- le Renard-Refacto
